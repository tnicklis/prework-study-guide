HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Tags start and end an element.
  • Closing tags have a "/" inside the carrots
  • Unordered lists or ul are bullet points; and ordered lists or ol will be numbered. List elements are denoted with "li".

CSS

  • Margin - how much space around the outside of an element
  • Padding - how much space around the content inside an element
  • CPBM - content, padding, border, margin (moving from inside to outside)
  • CSS created in an external style sheet that must be linked. Often this is done in the head element using a link.
  • Class atributes allow us to assign styling to specific elements. Whereas element names apply styling to every element with that type.
  • Assign class attributes in CSS with ".example" and in HTML by inserting class="example" into the first part of the element tag
  • Assigning colors in CSS with hex# can be found here.
