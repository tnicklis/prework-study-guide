HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Tags start and end an element.
  • Closing tags have a "/" inside the carrots
  • Unordered lists or ul are bullet points; and ordered lists or ol will be numbered. List elements are denoted with "li".

CSS

  • Margin - how much space around the outside of an element
  • Padding - how much space around the content inside an element
  • CPBM - content, padding, border, margin (moving from inside to outside)
  • CSS created in an external style sheet that must be linked. Often this is done in the head element using a link.
  • Class atributes allow us to assign styling to specific elements. Whereas element names apply styling to every element with that type.
  • Assign class attributes in CSS with ".example" and in HTML by inserting class="example" into the first part of the element tag
  • Assigning colors in CSS with hex# can be found here.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • Variables are containers that store values. Start by declaring a vaiable with the "let" keyword.
  • Data Types: String, Number Boolean, Array, Object
  • Operators: === is strict equality. Boolean test
  • !, !== is Not,Does-not-equal
  • Conditionals are if/else statements, with the test going inside the if()
  • Functions are a way of packaging functionality we wish to reuse
  • Events are for real interactivity on websites. Example: document.querySelector("html").addEventListener("click", () => { alert("Ouch! Stop poking me!");
  • JavaScript data types and data structures can be found here.
  • Add JS below footer element using script tags.
  • Falsy= value is 0, -0, On, "", null, undefined, NaN, or Boolean false.
  • Truthy= all other values and Boolean true. more info on truthy/falsy here.
  • Arrays are used to store set of data. Can be different types. Index starts counting at 0.
  • For Loops- predictable pattern of indices to perform a task on all items in the array.
